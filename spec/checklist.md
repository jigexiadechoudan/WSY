# Vision Mentor 验收清单

## 后端 (Backend)
- [ ] `/api/v1/vision/analyze-pose` 接口能正确接收和解析 MediaPipe 传来的 `landmarks` 数据。
- [ ] 坐标归一化算法能有效消除手在画面中不同位置和远近带来的误差。
- [ ] 手势相似度分数（0-100）计算稳定，同一种手势分数波动较小。
- [ ] 大模型服务（LLM）调用成功，能根据不同非遗场景生成对应风格的点评文字。

## 前端 (Frontend)
- [ ] `HandTracking.jsx` 能流畅捕获摄像头画面并提取骨骼点。
- [ ] `VisionMentor.jsx` 界面美观，符合整体 UI 风格（包含目标手势指引）。
- [ ] 实时分数进度条无明显卡顿或闪烁。
- [ ] “高分保持 2 秒”的逻辑判断准确，能正确触发成功状态。
- [ ] 成功后能清晰展示后端返回的 AI 点评信息。
- [ ] 能够自由切换不同的非遗场景（如苏绣、紫砂、皮影）。
