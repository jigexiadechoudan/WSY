# 创意艺匠 (Creative Artisan) - 云端非遗数字文创工坊

## 1. 核心理念与背景
原定的本地 Stable Diffusion 部署成本高、硬件依赖强，且易出现运行故障。为了“保证服务稳定可用”，并提供更深度的“非遗传承”体验，我们将“创意艺匠”模块升级为**“云端非遗数字文创工坊”**。
通过结合大语言模型（LLM）的对话/文学生成能力与云端文生图 API（如智谱 CogView / 阿里云通义万相 / OpenAI DALL-E 3），我们让用户在零门槛的情况下，不仅能创作出精美的非遗风格艺术品，还能与“虚拟非遗大师”对话，并为作品赋予文化内涵（如生成专属题诗与背景故事）。

## 2. 核心功能特性
- **AI 匠人对话引导 (Virtual Master Chat)**:
  用户输入一个简单的创意（如“我想画一只赛博朋克风格的猫”），系统背后的 LLM 扮演非遗传承人（如“苏绣大师”或“青花瓷匠人”），引导用户如何将该创意与非遗技艺结合，并自动扩写为高质量的绘画提示词（Prompt）。
- **云端非遗生图 (Cloud AIGC Generation)**:
  对接稳定的云端绘画 API（避免本地显卡限制）。传入经过大师优化的提示词（如：“赛博朋克机械猫，苏绣质感，丝线反光，精细针法...”），快速生成非遗风格的数字艺术品。
- **非遗数字藏品卡片 (Digital Artifact Card)**:
  图片生成后，再次调用 LLM，根据生成的画面和非遗类别，创作一首符合意境的中国古典诗词，并配以一段“文物鉴定/文化解说”文字，最终在前端生成一张精美、可分享的“数字藏品卡片”。

## 3. 技术架构与数据流
1. **用户创意输入**: 前端用户在“创意工坊”界面输入简短的想法（Idea），并选择非遗流派（如：苏绣、青花瓷、皮影戏）。
2. **LLM 提示词扩写**: 
   - 请求后端 `/api/v1/creative/enrich-prompt`
   - LLM 将简单的 Idea 转化为两部分：
     1. 给用户的匠人点评（如：“这个想法很大胆！如果用青花瓷的釉里红技法表现，会非常惊艳...”）
     2. 用于生图的精准英文 Prompt（包含特定的非遗质感词汇）。
3. **云端生图调用**:
   - 请求后端 `/api/v1/creative/generate-image`
   - 后端调用云端绘画 API（如 Zhipu API 或通义万相），获取生成的图片 URL。
4. **文化赋能 (Poem & Story)**:
   - 请求后端 `/api/v1/creative/generate-story`
   - LLM 为该图片生成专属标题、一首古诗、一段文化解读。
5. **前端卡片渲染**:
   - 将图片、诗词、解说组合，使用前端组件渲染成“数字藏品卡片”，并支持一键下载或分享。

## 4. 优势与扩展性
本方案无需用户或服务器配置昂贵的 GPU，API 调用轻量且 100% 稳定。相较于单纯的“画图工具”，它更像是一个**非遗文化体验与教育平台**。生成的“数字藏品卡片”具有极强的社交分享属性，能更好地促进非遗文化的传播与破圈。
