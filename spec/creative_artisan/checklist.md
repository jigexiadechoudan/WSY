# 创意艺匠 (Creative Artisan) 验收清单

## 后端 (Backend)
- [ ] **云端大模型 API 集成成功**（Zhipu/DashScope 等）：
  - `IMAGE_API_KEY` 等敏感信息不在代码仓库中明文存放，而是通过 `.env` 管理。
  - 能稳定调用 Text-to-Image 接口，并返回清晰的高清非遗风格图片 URL。
- [ ] `/api/v1/creative/enrich-prompt` 接口正常：
  - [ ] LLM 能够模拟“非遗传承人”的口吻与用户互动。
  - [ ] LLM 能够自动为简短的用户创意（Idea）添加符合该非遗流派（如“苏绣”、“青花瓷”、“皮影”）的专业材质和工艺描述词。
- [ ] `/api/v1/creative/generate-story` 接口正常：
  - [ ] 能够基于用户的创作生成一首符合意境的中国古典诗词（如五言/七言绝句）。
  - [ ] 能够生成具有文化内涵的文物/藏品背景故事介绍。
- [ ] 错误处理完善（如：API 超时、请求失败时能返回优雅的错误提示语）。

## 前端 (Frontend)
- [ ] **创意工坊页面 (`CreativeWorkshop.jsx`)**:
  - [ ] 提供清晰的输入区域，供用户输入创意（Idea）并选择非遗风格流派。
  - [ ] 展示与“AI 大师”的对话区域，让交互过程具有沉浸式的传承氛围。
  - [ ] 提供 Loading 状态反馈（由于生图和写诗可能需要几秒到十几秒不等）。
- [ ] **数字藏品卡片 (`ArtifactCard.jsx`)**:
  - [ ] 卡片排版精美，包含生成的非遗图片、标题、诗词、解说。
  - [ ] 卡片背景带有适当的中国风或复古纹理（如宣纸、卷轴质感）。
  - [ ] 卡片支持通过点击“保存到本地”或“下载”按钮，将其导出为一张图片（PNG/JPEG），方便用户分享至社交平台。
- [ ] **整体交互体验**:
  - [ ] 用户无需配置任何本地 GPU 环境或复杂参数（CFG, Steps, Sampler 全部由后端/云端包揽）。
  - [ ] 流程顺畅，从“灵感输入” -> “大师点评” -> “作品生成” -> “藏品赋能”，构成一个完整的非遗 AIGC 体验闭环。
