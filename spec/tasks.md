# Vision Mentor 实现任务拆解

### Task 1: 预置标准手势数据准备
- [ ] 编写一个简单的 Python 或前端脚本，用于录制并提取三个经典手势（苏绣捏针、紫砂拍泥、皮影飞鸟）的 21 个关键点数据。
- [ ] 将数据保存为后端的标准手势库格式（如 `standard_poses.json` ）。

### Task 2: 后端视觉比对算法实现
- [ ] 在 `vision_mentor.py` 的 `/analyze-pose` 接口中实现数据接收逻辑。
- [ ] 实现坐标归一化算法（以手腕点为原点，根据手掌大小进行缩放归一化）。
- [ ] 实现余弦相似度或欧氏距离计算函数，将相似度映射到 0-100 的分数。

### Task 3: AI 点评集成
- [ ] 在 `vision_mentor.py` 中引入 `llm_service`。
- [ ] 编写 prompt 模板，根据非遗场景（scenario）和分数（score）动态生成鼓励性点评。
- [ ] 将生成的点评文本随接口返回给前端。

### Task 4: 前端交互界面改造
- [ ] 修改 `VisionMentor.jsx`，增加目标手势的提示图层。
- [ ] 添加向后端发送 `landmarks` 数据的定时请求逻辑（使用 `fetch` 或 `axios`，注意防抖/节流）。
- [ ] 增加实时的环形进度条显示分数。
- [ ] 实现“保持高分 2 秒解锁”的交互逻辑。
- [ ] 弹窗或侧边栏展示 AI 点评内容。

### Task 5: 联调与测试
- [ ] 启动前后端服务进行整体链路测试。
- [ ] 调整相似度阈值和归一化参数，确保游戏体验不过于困难也不过于简单。
