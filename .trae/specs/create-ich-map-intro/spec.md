# 中国非遗入场页 (ICH Map Intro) Spec

## Why
系统需要一个充满人文气息的入场页，以此吸引用户并奠定整个中国非遗（Intangible Cultural Heritage）系统的情感基调。通过电影级镜头视角的地图穿梭效果，可以直观、震撼地展示中国各地的非遗文化传承与地理位置的联系，增强用户的代入感和文化认同感。

## What Changes
- 新增一个独立的入场页（Landing Page / Intro Page）。
- 引入 3D 地图或基于 WebGL/Canvas 的地图可视化组件（如 Three.js, ECharts-GL, 或 Mapbox 等）。
- 设计并实现电影镜头（Cinematic Camera）在地图上的平滑穿移与聚焦（Fly-to/Zoom-in）动画。
- 增加各地非遗项目的信息展示浮层或卡片（包括图片、视频、文字介绍），在镜头聚焦新地点时平滑展现。
- 增加充满人文气息的视觉元素（如水墨风、宣纸纹理、传统色彩、书法字体等）。
- 增加环境音效和非遗原声背景音乐（可选）。

## Impact
- Affected specs: 系统的初始路由入口将被更新为该入场页。
- Affected code: 前端路由配置，前端页面目录结构（新增 Intro 模块），全局样式与动画库依赖。

## ADDED Requirements
### Requirement: 地图漫游与非遗展示体验
系统 SHALL 提供一个全屏的交互式或自动漫游地图作为入场动画，并支持以下特性：

#### Scenario: 自动漫游展示
- **WHEN** 用户首次进入系统时
- **THEN** 页面播放一段流畅的地图穿梭动画，视角如电影镜头般从全景推进，依次聚焦到几个设定的中国非遗发源地（如：苏绣-苏州，皮影戏-陕西，唐卡-西藏）。
- **THEN** 每到一个地点，地图停止穿梭，展现该地非遗的高质量图文或视频卡片，配合符合人文气息的排版与转场动画。

#### Scenario: 交互与进入系统
- **WHEN** 漫游结束或用户点击“探索”/“跳过”按钮时
- **THEN** 平滑过渡并进入系统的主界面。
